
Usage:

    uberdeploy <command>

Commands:

    create		Create a new project locally and on remote
    init		Create a new project locally
    join 		Join an existing project
    update 		Update to latest version
    uninstall 		Uninstalls uberdeploy from your machine
    log			Displays log of current project

Options:

    -h | --help		Shows this page
    -v | --version	Shows current version
